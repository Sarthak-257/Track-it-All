## ðŸ› ï¸ Deploying the Net

### 1. Server Initialization (Express/MongoDB)
```bash
cd server
npm install
npm run build
npm start
```
**Required Environment Telemetry (.env):**
- `MONGODB_URI`: Secure connection string.
- `JWT_SECRET`: Authorization encryption key.
- `EMAIL_USER`: Primary uplink for OTP relay.
- `EMAIL_PASS`: Encrypted credential for mail server.

### 2. Client Activation (Next.js/Framer)
```bash
cd client
npm install
npm run build
npm start
```
**Required Environment Telemetry (.env.local):**
- `NEXT_PUBLIC_API_URL`: Backend endpoint coordinate.
